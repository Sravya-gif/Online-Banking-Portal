#  Banking Automation Framework

---

##  Project Overview

*This project is a Selenium-based Automation Testing Framework developed using Java and TestNG.
*It automates end-to-end functional testing of a demo online banking application.
*The framework follows the Page Object Model (POM) design pattern and is built as a reusable and scalable automation solution.

---

##  Features Covered

* Authentication
  
* Customer Management

* Account Management

* Transactions

* Reports

---

##  Framework Design

* Page Object Model (POM)
* TestNG for test execution
* WebDriverManager for driver setup
* ExtentReports for reporting
* Screenshot capture on failure
* Config-driven execution using `config.properties`

---

##  Reporting

* Extent Report generated after execution
* Screenshots captured on failure
* Reports available in `/reports` folder

---

##  Note

The Guru99 demo application may occasionally return **HTTP 500 errors** for certain modules like Edit Account and Reports.
