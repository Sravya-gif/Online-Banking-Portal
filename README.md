\# Banking Automation Framework



\##  Project Overview



This project is a Selenium-based Automation Testing Framework developed using Java and TestNG. It automates end-to-end functional testing of a demo online banking application.



The framework follows the \*\*Page Object Model (POM)\*\* design pattern and is built as a reusable, scalable test automation solution.

\##  Framework Design



\* Page Object Model (POM)

\* TestNG for test execution

\* WebDriverManager for driver setup

\* ExtentReports for reporting

\* Screenshot capture on failure

\* Config-driven execution (config.properties)



\---





\##  Reporting



\* Extent Report generated after execution

\* Screenshots captured on failure

\* Located in `/reports` folder
